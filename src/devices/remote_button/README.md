# Button

Simple Iot button with 3d printed parts.

## Functional Requirements

Button has following requirements:

- using MQTT for communication
- battery powered
  - not specified battery type lipo / Alkaine
- using esp12s
- on button pressed sends event on mqtt topic
- configurable through .json configuration
- event don't have to be send 'right away' even 20s delay is accepatble
- in case of failure - ignore command
